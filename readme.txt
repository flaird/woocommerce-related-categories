Define related categories and optionally display them under each product.

Very much based on Term Management Tools by scribu: https://wordpress.org/plugins/term-management-tools/.

== Installation ==

= From your WordPress dashboard =

Visit 'Plugins > Add New'
Search for and download 'woocommerce-related-categories'
Activate WooCommerce Related Categories from your Plugins page.

= From other sources =

Download WooCommerce Related Categories.
Upload the 'woocommerce-related-categories' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
Activate WooCommerce Related Categories from your Plugins page.
